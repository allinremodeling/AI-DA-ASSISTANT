## Secrets Supabase

| Secret | Uso |
|--------|-----|
| `ANTHROPIC_API_KEY` | **Recomendado** — orquestador principal (Claude) |
| `ANTHROPIC_CHAT_MODEL` | Opcional — default `claude-sonnet-4-20250514` |
| `OPENAI_API_KEY` | **Obligatorio** — visión + fallback orquestador |
| `TAVILY_API_KEY` | Inspiración web en vivo |

```powershell
npx supabase secrets set `
  ANTHROPIC_API_KEY=sk-ant-... `
  OPENAI_API_KEY=sk-... `
  TAVILY_API_KEY=tvly-...
npm run deploy:chat
```
