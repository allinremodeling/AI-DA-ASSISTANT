## Problemas conocidos / limitaciones

1. **cPanel desactualizado** — Si ves layout antiguo (4 pilares, slabs duplicados, mobile roto), sube el ZIP nuevo. El hash JS en producción (`BZu9LrRK`) ≠ build V1.5.0 (`B7yE_QLu`).

2. **SmartSlab API pública** — `https://smart-slab-app.vercel.app/api/listings` responde 404. El sistema usa:
   - Supabase SmartSlab si hay secrets, o
   - Lista fallback sincronizada (Calacatta Hudson, Irving, Plano, etc.)

3. **Modo demo local** — Sin `OPENAI_API_KEY` en Supabase, las respuestas son genéricas en español.

4. **Card 2 sin imagen** — Por diseño del Prompt Maestro: inspiración textual basada en web, no copia de imágenes.

5. **Edge Function no redeployada** — Cambios en `supabase/functions/` no aplican hasta `npm run deploy:chat`.
